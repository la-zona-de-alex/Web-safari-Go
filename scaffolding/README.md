# Scaffolding

Plantillas iniciales para arranque rápido:

- `scaffolding/frontend`: app React (Vite)
- `scaffolding/gateway`: gateway (proxy + auth)
- `scaffolding/backend`: backend básico con JWT
- `scaffolding/services/*`: microservicios minimalistas

Para probar rápidamente, instalar dependencias en cada carpeta y ejecutar sus scripts.
